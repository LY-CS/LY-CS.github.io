# CSharp基础·泛型

# 为什么要使用集合？

数组的局限性：

（1）数组元素个数是固定的，数组一但定义，就无法改变元素总数。如果需求变化，则必须修改源码

（2）如果初始化元素总数非常大，则会造成空间浪费。

集合的特点：

（1）根据需要动态增加元素个数，没有限制。

（2）可以用以存储多个对象，这是个十分重要的属性。

# 泛型List

List泛型集合的特点：

（1）表示泛型，T是Type的简写，表示当前不确定具体类型。

（2）可以根据用户的实际需要，确定当前集合需要存放的数据类型，一旦确定不可改变。

List泛型使用前的准备工作：

（1）引入命名空间:System.Collections.Generic

（2）确定存储类型：List students = new List();

常用方法：

            //创建泛型集合
            List<string> nameList = new List<string>();
            //添加string元素
            nameList.Add("张三");
            nameList.Add("李四");
            nameList.Add("王五");
            nameList.Add("林六");
            //创建方法2
            //List<string> nameList = new List<string>(){"张三","李四","王五","林六"}

            //统计泛型集合元素个数
            Console.WriteLine("元素总数：{0}", nameList.Count);

            //删除一个元素
            nameList.Remove("张三");
            Console.WriteLine("元素总数：{0}", nameList.Count);

            foreach (string item in nameList)
            {
                Console.WriteLine(item);
            }

# 泛型集合

关于Dictionary<K,V>泛型集合：

（1）Dictionary<K,V>通常称为字典，<K,V>约束集合中元素类型。

（2）编译时检查类型约束，无需装拆箱操作。

Dictionary<K,V>泛型集合存储结构

常用方法：

            //创建泛型集合
            Dictionary<string,string> nameList = new Dictionary<string, string>();
            //添加string元素
            nameList.Add("张三","5岁");
            nameList.Add("李四","6岁");
            nameList.Add("王五","7岁");
            nameList.Add("林六","8岁");

            //查找某个元素
            Console.WriteLine(nameList["张三"]);

            //遍历字典的key
            foreach(string key in nameList.Keys)
            {
                Console.WriteLine(key);
            }
            //遍历字典的value
            foreach (string value in nameList.Values)
            {
                Console.WriteLine(value);
            }
