# CSharp基础·字符串

# CSharp基础·字符串·特性

不可变性：不管对字符串做什么操作，都会在内存中产生一块新的实例。

驻留池（暂存池）：新建字符串并不会立刻开辟空间，而是查找是否有相同的字符串已经存在于系统，有则指向该字符串地址，无则开辟。

# CSharp基础·字符串·常用方法

IsNullOrEmpty(): 判断字符串是否为空
            string s = null;
            if (string.IsNullOrEmpty(s))
            {
                Console.WriteLine("是的");
            }
            else
            {
                Console.WriteLine("不是");
            }
            Console.ReadKey();

ToCharArray(): 字符串转换为字符数组
            string str = "abcdefg";
            char[] chs = str.ToCharArray();
            foreach(char item in chs)
            {
                Console.WriteLine(item);
            }
            Console.ReadKey();

            //char转换为字符串
            str = new string(chs);
            Console.ReadKey();

ToUpper()&ToLower()：转换字符串大小写
            string str = "abCdefG";
            Console.WriteLine(str.ToUpper());
            Console.WriteLine(str.ToLower());
            Console.ReadKey();

Equals：字符串比较
            string s1 = "abc";
            string s2 = "Abc";
            //忽略大小写
            if(s1.Equals(s2))
            {
                Console.WriteLine("相等");
            }
            else
            {
                Console.WriteLine("不相等");
            }
            Console.ReadKey();

indexof()：查找字符串出现的位置
            string str = "老毕吃了一顿饭，结果老毕活了";
            int index = str.IndexOf("老");
            Console.WriteLine(index);
            Console.ReadKey();
