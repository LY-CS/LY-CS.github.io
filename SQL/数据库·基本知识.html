# 数据库·基本知识

# 数据库·基本概念·实体

概念：客观存在的事物都是实体。

# 数据库·基本概念·数据库管理系统

概念：是一个由用户使用的数据库管理软件，目的就是让用户通过操作数据库管理系统完成对数据库数据的处理。 功能：

（1）创建数据库：能够提供数据定义语言（Data Defination Lang，简称DDL），用户利用DDL可以方便地建立数据库（数据表、关系、各种对象）。

（2）操作数据库：DBMS通过提供数据库操纵语言（Data Manipulation Language，简称DML）实现其数据地插入、修改、删除、查询、统计等数据存取及处理功能。

# 数据库·基本概念·T-SQL

概念：是SQL的升级，对功能进行了扩充，如变量说明、流程控制、功能函数。

组成：

（1）DDL（数据定义语言）

（2）DML（数据操作语言）

（3）DCL（数据控制语言）：用来控制存取许可、存取权限等。

（4）变量说明、流程控制、功能函数

# 数据库·基本概念·主流数据库

Oracle：产品免费、服务收费，适合大、中型企业使用。

SQLServer：针对不同用户群体的多个版本。

DB2：支持多操作系统、多种类型的硬件和设备。

MySQL：开放源代码、跨平台性能好。在网站中应用广泛。
